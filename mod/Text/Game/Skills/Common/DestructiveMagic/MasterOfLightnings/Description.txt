The hero learns the Lightning Bolt spell, and is granted stunning effect to Lightning Bolt and Chain Lightning spells.<br><color_default>The ATB value of the first target hit is multiplied by 0.5, therefore the effect is more powerful against targets that are about to act. For example, an ATB value of 0.8 gets reduced to 0.4, while an ATB value of 0.1 gets reduced to 0.05.
